# Создание ботов для Telegram и VK

## Telegram

### 1. Создать бота

1. Открыть [@BotFather](https://t.me/BotFather) в Telegram
2. Отправить `/newbot`
3. Ввести имя бота (отображаемое): например, `Poll Aggregator`
4. Ввести username бота (уникальный, заканчивается на `bot`): например, `my_poll_aggregator_bot`
5. BotFather вернёт токен вида `123456789:AABBccDD-EeFfGgHhIiJjKkLlMmNnOoPp`
6. Записать токен в `.env` как `TELEGRAM_BOT_TOKEN`

### 2. Настроить бота

Отправить BotFather следующие команды:

```
/setdescription — описание бота
/setabouttext — текст "О боте"
/setcommands — список команд:
  newpoll - Создать новый опрос
  status - Статус текущих опросов
```

### 3. Добавить бота в чат/канал

- Добавить бота в нужный чат как администратора
- Для каналов: добавить бота как администратора с правом отправки сообщений
- Бот должен иметь права на отправку опросов и редактирование сообщений

### 4. Получить chat_id

Самый простой способ:
1. Добавить бота в чат
2. Отправить любое сообщение в чат
3. Открыть `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Найти `"chat":{"id": -1001234567890}` — это `chat_id`

## VK

### 1. Создать сообщество

1. Перейти на [vk.com/groups](https://vk.com/groups) → «Создать сообщество»
2. Выбрать тип (например, «Группа по интересам»)
3. Указать название и тематику

### 2. Получить токен сообщества

1. Перейти в **Управление** → **Работа с API**
2. Вкладка **Ключи доступа** → «Создать ключ»
3. Отметить права:
   - Управление сообществом
   - Отправка сообщений
   - Доступ к стене
4. Скопировать ключ → записать в `.env` как `VK_GROUP_TOKEN`

### 3. Получить ID группы

- ID группы видно в URL: `vk.com/club123456789` → ID = `123456789`
- Или: **Управление** → **Информация** → внизу страницы
- Записать в `.env` как `VK_GROUP_ID`

### 4. Настроить Callback API

1. **Управление** → **Работа с API** → вкладка **Callback API**
2. Указать URL сервера: `https://your-server.com/webhook/vk`
3. Скопировать строку подтверждения → записать в `.env` как `VK_CONFIRMATION_STRING`
4. После запуска сервера — нажать «Подтвердить»
5. Перейти на вкладку **Типы событий**, включить:
   - `poll_vote_new` — голос в опросе

### 5. Включить Long Poll (резерв)

Если Callback API не подходит:
1. **Управление** → **Работа с API** → вкладка **Long Poll API**
2. Включить Long Poll API, версия 5.199+
3. Включить событие `poll_vote_new`

## Проверка

После заполнения `.env` — запустить приложение и проверить:

```bash
# Проверить Telegram-токен
curl https://api.telegram.org/bot$TELEGRAM_BOT_TOKEN/getMe

# Проверить VK-токен
curl "https://api.vk.com/method/groups.getById?access_token=$VK_GROUP_TOKEN&v=5.199"
```
