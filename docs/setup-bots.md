# Создание ботов для Telegram и VK

## Telegram

### 1. Создать бота

1. Открыть [@BotFather](https://t.me/BotFather) в Telegram
2. Отправить `/newbot`
3. Ввести имя бота (отображаемое): например, `Poll Aggregator`
4. Ввести username бота (уникальный, заканчивается на `bot`): например, `my_poll_aggregator_bot`
5. BotFather вернёт токен вида `123456789:AABBccDD-EeFfGgHhIiJjKkLlMmNnOoPp`
6. Записать токен в `.env` как `TELEGRAM_BOT_TOKEN`

### 2. Настроить бота

Отправить BotFather следующие команды:

```
/setdescription — описание бота
/setabouttext — текст "О боте"
/setcommands — список команд:
  newpoll - Создать новый опрос
  status - Статус текущих опросов
```

### 3. Добавить бота в чат/канал

- Добавить бота в нужный чат как администратора
- Для каналов: добавить бота как администратора с правом отправки сообщений
- Бот должен иметь права на отправку опросов и редактирование сообщений

### 4. Получить chat_id

Самый простой способ:
1. Добавить бота в чат
2. Отправить любое сообщение в чат
3. Открыть `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Найти `"chat":{"id": -1001234567890}` — это `chat_id`

## VK

VK работает через OAuth — любой админ группы может подключить бота к своей группе через веб-интерфейс.

### 1. Создать VK-приложение (одноразово, для владельца бота)

Приложение создаётся один раз — через него все пользователи будут авторизовываться и подключать свои группы.

#### 1.1. Регистрация

1. Перейти на [vk.com/apps?act=manage](https://vk.com/apps?act=manage)
2. Нажать **«Создать»**
3. Заполнить:
   - **Название**: например, `Poll Aggregator`
   - **Платформа**: выбрать **Standalone-приложение**
4. Нажать **«Подключить приложение»**
5. Подтвердить через SMS-код, если потребуется

#### 1.2. Настройка приложения

После создания — перейти на вкладку **«Настройки»**:

1. **ID приложения** (числовой, вверху страницы) → записать в `.env` как `VK_APP_ID`
2. **Защищённый ключ** → записать в `.env` как `VK_APP_SECRET`
3. Заполнить поля:
   - **Адрес сайта**: `https://your-server.com` (ваш домен с HTTPS)
   - **Базовый домен**: `your-server.com` (без протокола)
   - **Доверенный redirect URI**: `https://your-server.com/vk/callback`
4. Нажать **«Сохранить»**

> **Важно:** redirect URI должен точно совпадать с тем, что указан в `WEBHOOK_BASE_URL` + `/vk/callback`. Если URL не совпадёт, VK вернёт ошибку при OAuth-авторизации.

#### 1.3. Включить приложение

1. Перейти на вкладку **«Информация»**
2. Убедиться, что статус приложения — **«Приложение включено и видно всем»**
3. Если приложение выключено — включить его

> Пока приложение в статусе «Выключено» или «На модерации», OAuth-авторизация будет работать только для администраторов приложения.

### 2. Сгенерировать ключ шифрования

Токены групп хранятся в БД зашифрованными. Сгенерировать ключ:

```bash
python -c 'from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())'
```

Записать результат в `.env` как `VK_TOKEN_ENCRYPTION_KEY`.

### 3. Подключение группы (для каждого пользователя)

1. Пользователь переходит по ссылке: `https://your-server.com/vk/connect`
2. Авторизуется через VK, выдаёт права на управление группами
3. Выбирает группу из списка своих (где он админ)
4. Бот автоматически:
   - Сохраняет зашифрованный токен
   - Настраивает Callback API для группы
   - Включает событие `poll_vote_new`
5. Готово — можно создавать опросы в этой группе

### 4. Отключение группы

```bash
curl -X POST https://your-server.com/vk/disconnect/{group_id}
```

## Проверка

После заполнения `.env` — запустить приложение и проверить:

```bash
# Проверить Telegram-токен
curl https://api.telegram.org/bot$TELEGRAM_BOT_TOKEN/getMe

# Проверить VK OAuth (открыть в браузере)
# https://your-server.com/vk/connect
```
